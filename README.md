# MDINotepad
MDI Notepad Use Windows Forms App (.NET Framework) and has one Form use bunifu framework.
Funtion: 
  File:
        + New
        + Open
        + Save
        + Save As
  Edit:
        + Undo
        + Redo
        + Copy
        + Cut
        + Paste
  Format:
        + WordWrap
        + Font
        + Color
  View: 
        + Zoom (In, Out)
        + Status Bar
        + Arrange Icons
        + Cascade
        + Title Horizontal
        + Title Vertical
  Help:
        + View Help
        + Send FeedBack
        + About Notepad
  Use MDI to build mutiple form richtextbox, and use strip bar, menu bar,...
